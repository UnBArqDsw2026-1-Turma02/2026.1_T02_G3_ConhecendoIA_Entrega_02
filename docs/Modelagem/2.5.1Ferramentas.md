# 2.5.1 Ferramentas

## Introdução

Durante o ciclo de concepção, planejamento e desenvolvimento do projeto **ConhecendoIA**, a equipe fez uso de um conjunto de ferramentas fundamentais para garantir uma comunicação eficiente, organização de cronogramas, modelagem do design e versionamento de código e documentação. Essa seleção de tecnologias permitiu a adoção plena de metodologias ágeis e do trabalho colaborativo remoto, reduzindo atritos na documentação e facilitando a evolução do software.

Abaixo estão listadas detalhadamente as principais ferramentas adotadas pela equipe e como contribuíram na prática:

| Ferramenta | Propósito no Projeto |
| :--- | :--- |
| **Git** | Utilizado para o controle de versão do código-fonte e da documentação. Ele permite o registro histórico de todas as alterações feitas no projeto, facilitando o trabalho paralelo por meio de branches sem que o código de um membro sobrescreva o de outro. |
| **GitHub** | Plataforma de hospedagem do repositório Git. Serve como ambiente central para nossa gestão de código, colaboração assíncrona, integração contínua (como hospedagem do GitHub Pages) e submissão de alterações por meio de *Pull Requests*. |
| **VSCode** *(Visual Studio Code)* | O ambiente de desenvolvimento integrado (IDE) principal escolhido devido à sua leveza, vasta comunidade de extensões e forte capacidade para lidar tanto com código nativo quanto com a edição estilizada de arquivos Markdown (`.md`). |
| **Discord** | Ferramenta primária para a comunicação de voz, sendo o ponto de encontro de sincronização da equipe (Daily Scrums, Plannings e Retrospectives) e garantindo também um local para compartilhamento de telas em reuniões de pareamento ou validações em tempo-real. |
| **When2meet** | Ferramenta auxiliar extremamente crítica para cruzar cronogramas e cruzar horários livres de todos os membros do grupo, garantindo um agendamento rápido e assertivo das nossas sprints e ritos ágeis. |
| **WhatsApp** | Canal de comunicação instantânea, onde são passados avisos urgentes, enquetes de decisões ágeis pontuais e links de acesso fora das cerimônias oficiais delimitadas pelo Discord. |
| **Draw.Io** | Ferramenta de naveador utilizada para criar diagramas, fluxogramas, mapas mentais, topologias de rede e organogramas de forma intuitiva e utilizada principalmente para a modelação dos diagramas. |
| **Lucidchard** | Ferramenta de diagramação inteligente baseada em nuvem que permite a criação colaborativa de fluxogramas, mapas de processos, diagramas UML, wireframes e organogramas em tempo real, também utilizada para a modelação dos diagramas. |
