# 2.4 Participações - Base

## Introdução

Este documento tem como objetivo relatar as principais contribuições e participações de cada membro da equipe no desenvolvimento dos artefatos da etapa base do projeto ConhecendoIA.

As participações foram organizadas em três focos principais:

-  Modelagem Estática (Foco 1)  
- Modelagem Dinâmica (Foco 2)  
- Modelagem Organizacional (Foco 3)  

Para cada foco, são apresentadas tabelas contendo o nome do membro, suas contribuições, a significância da contribuição e os respectivos elos comprobatórios.
