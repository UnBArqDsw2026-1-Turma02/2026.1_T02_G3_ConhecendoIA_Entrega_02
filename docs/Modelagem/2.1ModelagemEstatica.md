# 2.1 Modelagem Estática

## Introdução
A modelagem estática define a estrutura fundamental do sistema **ConhecendoIA**, estabelecendo como os componentes de software e as camadas de dados são organizados e interconectados. Enquanto a modelagem dinâmica foca no comportamento e no tempo, esta seção dedica-se a apresentar o "esqueleto" arquitetural que sustenta a plataforma, garantindo modularidade e escalabilidade.

## Comentários sobre o Trabalho em Equipe
O grupo demonstrou maturidade na aplicação das metodologias ágeis, mantendo um fluxo constante de comunicação para alinhar os diagramas às Histórias de Usuário. A transição para o uso de ferramentas de alta fidelidade visual permitiu uma padronização rigorosa, facilitando as discussões técnicas e a identificação antecipada de possíveis gargalos na implementação do backend.
