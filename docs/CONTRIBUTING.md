## Como Contribuir

1. **Faça um Fork** do projeto.
2. **Crie uma Branch** para sua Feature:
   ```bash
   git checkout -b feature/SuaFeature
   ```
3. **Faça o Commit** de suas mudanças:
   ```bash
   git commit -m 'Add: Descrição da nova funcionalidade'
   ```
4. **Faça o Push** para a Branch:
   ```bash
   git push origin feature/SuaFeature
   ```
5. **Abra um Pull Request**.

### O que você pode reportar ou sugerir?
* Correção de bugs ou melhorias de UI/UX.
* Otimização de performance no carregamento das páginas.
* Novas sugestões de categorias para o fórum.
